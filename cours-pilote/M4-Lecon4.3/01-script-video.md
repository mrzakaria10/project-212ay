# Leçon 4.3 — Construire un workflow complet
## Script Vidéo — ~12 min

### [0:00-0:30] HOOK
> Tu as passé 4 modules à apprendre à utiliser l'IA. Maintenant, on va tout assembler dans un workflow quotidien que tu peux appliquer dès demain.

### [0:30-3:00] CONCEPT — Le workflow idéal

**(Slide)**
```
1. MATIN (30 min)   → Planifier avec Claude
2. MATIN (2h)       → Coder avec Copilot
3. MIDI             → Debug avec Cursor
4. APRÈS-MIDI (1h)  → Tests + Doc avec ChatGPT
5. SOIR (30 min)    → Revue + Commit
```

### [3:00-8:30] DÉMO — Assemblage final

> On reprend le projet de l'API Task du Module 1 et on applique tout :

**Étape 1 — Claude :** Planifier l'architecture d'une nouvelle feature
**Étape 2 — Copilot :** Coder les endpoints dans VS Code
**Étape 3 — Cursor :** Corriger un bug de validation
**Étape 4 — ChatGPT :** Générer la doc + tests
**Étape 5 — Revue :** Relire le code généré, valider

### [8:30-9:00] ACTION FINALE DU COURS
> Ta mission finale : prends un mini-projet et exécute ce workflow complet. Note le temps total. Compare avec un projet fait sans IA.

### [9:00-9:30] CONCLUSION DU COURS
> Félicitations ! Tu as terminé les 4 modules. Tu sais maintenant utiliser l'IA comme un vrai copilote de développement. N'oublie pas : l'IA écrit le code, toi tu décides ce qui est bon. Va coder, et surtout : vibe coding !
