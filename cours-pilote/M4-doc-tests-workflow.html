Hook

Tu finis un projet, tu remets la doc à plus tard — et dans 3 mois, plus personne ne comprend le code. L'IA peut générer une doc complète en 30 secondes.

Concept — 3 types de doc

README — présentation, installation, endpoints
JSDoc/TSDoc — documentation inline des fonctions
OpenAPI — spec complète des routes

Démo

"Génère un README.md pour mon API Express avec installation, endpoints, exemples." → README complet en 10 sec.

"Génère une spec OpenAPI 3.0 pour ces routes" → YAML prêt à l'emploi.

Action

Prends un projet. Colle le code dans ChatGPT. Demande un README complet. Publie-le.
