# Leçon 2.2 : Générer du boilerplate utile
## Version Texte HCDA

## H — Hook
Tu passes 15 minutes à créer la structure d'un nouveau projet. Et si tu pouvais tout avoir en 30 secondes ?

## C — Concept
L'IA excelle dans les tâches **répétitives et prévisibles** : structures de projet, composants standards, routes CRUD, fichiers de config. Ce sont des patterns qu'elle a vus des milliers de fois.

**Use cases idéaux :**
- Structure Next.js / Express complète
- Composants : Card, Modal, Button, Layout
- Routes CRUD prêtes à l'emploi
- Config : ESLint, tsconfig, tailwind.config

## D — Démo

**Use case 1 — Structure Next.js :**
Prompt : "Génère la structure complète d'un projet Next.js 14 avec app router, layout, Header/Footer, Tailwind, dossier lib/ et types/"
→ Squelette complet en un prompt.

**Use case 2 — Modal réutilisable :**
Prompt : "Composant Modal React + Tailwind avec overlay, fermeture au clic, animation fade, createPortal, TypeScript"
→ Composant prêt à l'emploi.

**Use case 3 — CRUD Express :**
Prompt : "Routes Express CRUD produits avec TypeScript, validation Zod, recherche, pagination"

## A — Action
Choisis un boilerplate dont tu as besoin. Génère-le avec un prompt structuré. Note le temps gagné.
